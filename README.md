# Agentic Joke Bot using LangGraph

This project demonstrates agentic workflow orchestration
using LangGraph without Large Language Models.

## Features
- Explicit state management
- Conditional routing
- Human-in-the-loop interaction
- Multilingual support
- Resettable memory

## Technologies
- Python
- LangGraph
- PyJokes
- Pydantic

## Run
pip install -r requirements.txt
python langgraph_joke_bot.py
